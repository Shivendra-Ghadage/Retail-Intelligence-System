## About the Project

This project is an end-to-end Retail Business Intelligence and Predictive Analytics system developed using Python, SQL, and Power BI.

The main goal of this project is to analyze large retail sales data (200,000+ records) and extract meaningful insights to help businesses make better decisions.

In this project, I worked on the complete data analytics lifecycle ‚Äî from data cleaning to machine learning and dashboard development.


## Project Objectives

* Analyze retail sales performance
* Predict future sales using time-series forecasting
* Identify customers who are likely to churn
* Analyze Customer Lifetime Value (CLV)
* Evaluate marketing channel performance
* Study return rate and operational efficiency
